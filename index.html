Lesson One

Setup.

Before you start coding you need to know the correct way to set up a HTML file. To setup a HTML file you need to follow theese steps.
THE CODE:


<!DOCTYPE html>
<html>
<head>
<title>ENTER YOUR TITLE</title> <style>
Here is where you enter CSS code to change size of text. </style>
</head>
<body>
Here is where you enter your text. </body>
</html>

Lesson Two

The basic text.

Text's are really simple and eazy to create. Firstly to create a Tittle or Header of your webpage. To create a Header you need to use the element h1. It looks like this:
<h1>Hello World!</h1>

Now to make a paragraph you can use the p element. This looks like this:
<p>Hello World!</p>
This is how I created this line of text here.


Now you can add other stuff but that will be in a later lesson. Your h1 element is from 1-6 the change the size of it 6 being the smallest 1 being the biggest you just change the number. You can also chnage it in the style tab which will be explained later through the lessons.
